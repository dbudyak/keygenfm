# Changelog

All notable changes to Keygen FM will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.1] - 2026-03-13

### Added
- Initial F-Droid release
- 24/7 chiptune radio streaming
- Windows 95-inspired retro UI
- Background playback with MediaSessionService
- Real-time now playing metadata (track, artist, album art)
- Volume control
- Live listener count display
- Persistent media notification with playback controls

### Technical
- Built with Jetpack Compose and Material3
- ExoPlayer/Media3 for audio streaming
- Hilt for dependency injection
- Retrofit for AzuraCast API integration
- Minimum SDK: 26 (Android 8.0)
- Target SDK: 35
