# Keygen FM

A 24/7 chiptune, cracktro, and demoscene internet radio station. Stream at [keygen-fm.kodatek.app](https://keygen-fm.kodatek.app)

## Apps

### Android

A retro Windows 95-styled radio player built with Kotlin and Jetpack Compose.

[<img src="https://fdroid.gitlab.io/artwork/badge/get-it-on.png"
     alt="Get it on F-Droid"
     height="80">](https://f-droid.org/packages/app.kodatek.keygenfm/)

**Features:**
- 24/7 streaming
- Now playing metadata with album art
- Background playback
- Volume control
- Live listener count
- Classic Windows 95 UI

See [android/README.md](android/README.md) for build instructions.

## Radio Backend

The station runs on **AzuraCast** (self-hosted):
- **Stream:** `https://keygen-fm.kodatek.app/listen/keygen-fm/radio.mp3`
- **Now Playing API:** `https://keygen-fm.kodatek.app/api/nowplaying/keygen-fm`
- **Admin:** `https://keygen-fm.kodatek.app`
